問題 1単振り子の線形化と非線形補正
物理的文脈：振り子の運動方程式 — 量子論・非線形力学の出発点。

長さ \(L\)、質量 \(m\) の単振り子の運動方程式は：

\[\ddot{\theta} + \frac{g}{L}\sin\theta = 0\]

（1）

\(\theta \ll 1\) の近似（\(\sin\theta \approx \theta\)）で方程式を線形化し、一般解と周期 \(T_0\) を求めよ。

（2）

初期条件 \(\theta(0) = \theta_0,\; \dot{\theta}(0) = 0\) での特殊解を求めよ。

（3）

\(\sin\theta = \theta - \theta^3/6 + \cdots\) の次の近似として \(\theta = \theta_0\cos\omega_0 t + \varepsilon\theta_1(t)\) と展開し（\(\varepsilon = \theta_0^2/6\)）、\(\theta_1\) が満たす方程式を導け。久期項（共鳴項）の除去から補正された周期 \(T = T_0\left(1 + \dfrac{\theta_0^2}{16}\right)\) を示せ。

ヒント： \(\cos^3\omega t = \dfrac{3}{4}\cos\omega t + \dfrac{1}{4}\cos 3\omega t\) を利用せよ。
問題 2減衰振動の3つのレジーム
物理的文脈：機械振動の制振設計・回路のQ値設計。

方程式 \(\ddot{x} + 2\gamma\dot{x} + \omega_0^2 x = 0\)（\(\gamma, \omega_0 > 0\)）について答えよ。

（1）過減衰 \(\gamma > \omega_0\)

一般解を求めよ。初期条件 \(x(0)=x_0,\;\dot{x}(0)=0\) での特解を書け。

（2）臨界減衰 \(\gamma = \omega_0\)

一般解を求め、同じ初期条件での特解を示せ。これが過減衰・不足減衰の中で「最も速く平衡に戻る」ことを定性的に議論せよ。

（3）不足減衰 \(\gamma < \omega_0\)

準固有角振動数 \(\omega_1 = \sqrt{\omega_0^2 - \gamma^2}\) を使って一般解を書き、振幅が \(e^{-\gamma t}\) で減衰することを示せ。

（4）

品質因数 \(Q = \omega_0/(2\gamma)\) を定義し、不足減衰で \(Q\) サイクルの間にエネルギーが \(e^{-2\pi}\) 倍になることを示せ（\(Q \gg 1\) の近似で）。

ヒント： エネルギー \(E \propto A^2 \propto e^{-2\gamma t}\)。
問題 3強制振動と共鳴
物理的文脈：LCR回路・地震応答・NMR。

方程式 \(\ddot{x} + 2\gamma\dot{x} + \omega_0^2 x = \dfrac{F_0}{m}\cos\omega t\) について答えよ。

（1）

定常特解を \(x_p = A(\omega)\cos(\omega t - \delta)\) の形で求め、振幅 \(A(\omega)\) と位相遅れ \(\delta(\omega)\) を導出せよ：

\[A(\omega) = \frac{F_0/m}{\sqrt{(\omega_0^2-\omega^2)^2 + 4\gamma^2\omega^2}}, \qquad \tan\delta = \frac{2\gamma\omega}{\omega_0^2-\omega^2}\]

（2）

\(A(\omega)\) を最大にする駆動角振動数 \(\omega_{\rm res}\) を求めよ。

（3）非減衰共鳴 \(\gamma=0, \omega=\omega_0\)

特解が \(x_p = \dfrac{F_0}{2m\omega_0}t\sin\omega_0 t\) となることを未定係数法で示せ。

ヒント： (1) 複素数を使い \(x = \mathrm{Re}[Ce^{i\omega t}]\) として \(C\) を求める方法が計算しやすい。
問題 4連立ODE — 連成振動子の正規モード
物理的文脈：分子振動・格子振動・結合振動子の基礎。

2個の質量 \(m\) が3本のバネ（バネ定数 \(k\)，中央バネは \(\kappa\)）で結ばれた系の運動方程式：

\[\begin{cases}m\ddot{x}_1 = -kx_1 - \kappa(x_1-x_2)\\ m\ddot{x}_2 = -kx_2 + \kappa(x_1-x_2)\end{cases}\]

（1）

行列形式 \(m\ddot{\mathbf{x}} = -K\mathbf{x}\) に書き直し、\(K\) を明示せよ。

（2）

\(\mathbf{x} = \mathbf{a}\,e^{i\omega t}\) と仮定して固有値問題 \(K\mathbf{a} = m\omega^2\mathbf{a}\) を解き、2つの正規振動数 \(\omega_\pm\) を求めよ。

（3）

各正規モードの固有ベクトルを求め、それぞれの振動の物理的意味（同相・逆相）を説明せよ。

ヒント： \(\omega_- = \sqrt{k/m}\)（重心運動），\(\omega_+ = \sqrt{(k+2\kappa)/m}\)（相対振動）。
