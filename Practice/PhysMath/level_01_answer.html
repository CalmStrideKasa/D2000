解答 1スカラー場の勾配と方向微分

（1）勾配 \(\nabla\phi\) の計算

各成分を偏微分する：

\[\frac{\partial\phi}{\partial x} = 2xy,\quad \frac{\partial\phi}{\partial y} = x^2 + z^2,\quad \frac{\partial\phi}{\partial z} = 2yz\]

点 \(P=(1,2,-1)\) を代入：

\[\nabla\phi\Big|_P = (2\cdot1\cdot2,\;1^2+(-1)^2,\;2\cdot2\cdot(-1)) = (4,\,2,\,-4)\]

（2）方向微分

\[D_\mathbf{n}\phi = \nabla\phi\cdot\mathbf{n} = (4,2,-4)\cdot\frac{1}{\sqrt{3}}(1,1,1) = \frac{4+2-4}{\sqrt{3}} = \frac{2}{\sqrt{3}} = \frac{2\sqrt{3}}{3}\]

（3）最大変化方向

\(\phi\) が最も急増する方向は \(\nabla\phi\) 自身の方向：

\[\mathbf{n}_{\max} = \frac{\nabla\phi}{|\nabla\phi|} = \frac{(4,2,-4)}{\sqrt{16+4+16}} = \frac{(4,2,-4)}{6} = \left(\frac{2}{3},\frac{1}{3},-\frac{2}{3}\right)\]

最大変化率は \(|\nabla\phi|_P = 6\)。

物理的意味： \(\nabla\phi\) は等ポテンシャル面に垂直な方向を指し、その大きさがその点での場の「急峻さ」を表す。
解答 2電場の発散とGaussの法則

（1）発散

\[\nabla\cdot\mathbf{E} = \frac{\partial}{\partial x}(2xz) + \frac{\partial}{\partial y}(-y^2) + \frac{\partial}{\partial z}(y^2z) = 2z - 2y + y^2\]

（2）電荷密度（点 \((1,1,1)\)）

\[\rho = \varepsilon_0\,\nabla\cdot\mathbf{E}\Big|_{(1,1,1)} = \varepsilon_0(2\cdot1 - 2\cdot1 + 1^2) = \varepsilon_0\]

（3）Gaussの定理の確認

体積積分（\(0\le x,y,z\le 2\)）：

\begin{align} \iiint_V (2z-2y+y^2)\,dV &= \int_0^2\!\int_0^2\!\int_0^2 (2z-2y+y^2)\,dx\,dy\,dz \\ &= 2\int_0^2\!\int_0^2 (2z-2y+y^2)\,dy\,dz \\ &= 2\int_0^2 \left[2zy - y^2 + \frac{y^3}{3}\right]_0^2 dz \\ &= 2\int_0^2 \left(4z - 4 + \frac{8}{3}\right)dz \\ &= 2\left[2z^2 - 4z + \frac{8}{3}z\right]_0^2 \\ &= 2\left(8 - 8 + \frac{16}{3}\right) = \frac{32}{3} \end{align}
結果： 面積分も同じ \(32/3\) になる（確認は各面で \(\mathbf{E}\cdot\hat{n}\,dS\) を積分）。Gaussの定理が成立。
解答 3磁場の回転とStokesの定理

（1）回転

\[\nabla\times\mathbf{F} = \begin{vmatrix}\mathbf{e}_x&\mathbf{e}_y&\mathbf{e}_z\\\partial_x&\partial_y&\partial_z\\-y&x&0\end{vmatrix} = \mathbf{e}_x(0-0) - \mathbf{e}_y(0-0) + \mathbf{e}_z(1-(-1)) = 2\mathbf{e}_z\]

（2）Stokesの定理の両辺

右辺（面積分）： \(d\mathbf{S} = dS\,\mathbf{e}_z\)、\((\nabla\times\mathbf{F})\cdot\mathbf{e}_z = 2\)

\[\iint_S (\nabla\times\mathbf{F})\cdot d\mathbf{S} = 2\pi R^2\]

左辺（線積分）： \(x=R\cos\theta,\,y=R\sin\theta\) とすると \(d\mathbf{l} = (-R\sin\theta,R\cos\theta,0)d\theta\)

\begin{align} \oint_C\mathbf{F}\cdot d\mathbf{l} &= \int_0^{2\pi}(-R\sin\theta,R\cos\theta,0)\cdot(-R\sin\theta,R\cos\theta,0)\,d\theta \\ &= \int_0^{2\pi}(R^2\sin^2\theta + R^2\cos^2\theta)\,d\theta = \int_0^{2\pi} R^2\,d\theta = 2\pi R^2 \end{align}

両辺一致 ✓

（3）剛体回転との対応

\(\boldsymbol{\omega} = \omega\mathbf{e}_z\)，\(\mathbf{r} = (x,y,z)\) として：

\[\mathbf{v} = \boldsymbol{\omega}\times\mathbf{r} = (\omega e_z)\times(x,y,z) = (-\omega y,\,\omega x,\,0)\]

\(\omega = 1\) の場合これは \(\mathbf{F}\) に一致。また \(\nabla\times\mathbf{v} = 2\omega\mathbf{e}_z = 2\boldsymbol{\omega}\) ✓

物理的意味： 渦度 \(\boldsymbol{\xi} = \nabla\times\mathbf{v} = 2\boldsymbol{\omega}\) は剛体回転の角速度の2倍。
解答 4重力ポテンシャルとLaplacian

（1）勾配（重力加速度）

\(\dfrac{\partial r}{\partial x} = \dfrac{x}{r}\) を使って：

\[\frac{\partial\phi}{\partial x} = \frac{\partial}{\partial x}\left(-\frac{GM}{r}\right) = \frac{GM}{r^2}\cdot\frac{x}{r} = \frac{GMx}{r^3}\]

同様に \(y,z\) 成分も計算すると：

\[\nabla\phi = \frac{GM}{r^3}(x,y,z) = \frac{GM}{r^3}\mathbf{r}\] \[\therefore\quad \mathbf{g} = -\nabla\phi = -\frac{GM}{r^3}\mathbf{r}\quad\checkmark\]

（2）Laplacian（\(r>0\)）

球対称関数 \(f(r)=-GM/r\) に対して \(f'(r)=GM/r^2,\;f''(r)=-2GM/r^3\) なので：

\[\nabla^2\phi = f''(r)+\frac{2}{r}f'(r) = -\frac{2GM}{r^3}+\frac{2GM}{r^3} = 0\quad (r>0)\quad\checkmark\]

（3）\(r^n\) のLaplacian

\(f(r)=r^n\) に対して \(f'=nr^{n-1},\;f''=n(n-1)r^{n-2}\) より：

\[\nabla^2(r^n) = n(n-1)r^{n-2} + \frac{2}{r}\cdot nr^{n-1} = n(n-1)r^{n-2}+2nr^{n-2} = n(n+1)r^{n-2}\quad\checkmark\]

\(n=-1\) の場合 \(n(n+1)=(-1)(0)=0\) となり (2) と一致 ✓
