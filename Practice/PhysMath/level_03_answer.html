解答 1慣性テンソルの対角化と主軸

（1）重心

面積 \(S=ab/2\)。重心：\(x_G = a/3,\; y_G = b/3\)

（2）原点周りの慣性テンソル

面密度 \(\sigma = M/S = 2M/(ab)\)。積分領域 \(0\le x\le a,\;0\le y\le b(1-x/a)\)：

\begin{align} I_{xx}^{(0)} &= \sigma\int\!\!\int y^2\,dA = \sigma\int_0^a \frac{b^3}{3}\left(1-\frac{x}{a}\right)^3 dx = \sigma\cdot\frac{b^3}{3}\cdot\frac{a}{4} = \frac{Mb^2}{6}\\ I_{yy}^{(0)} &= \frac{Ma^2}{6}, \qquad I_{xy}^{(0)} = -\sigma\int\!\!\int xy\,dA = -\frac{Mab}{12} \end{align}

（3）重心周り（平行軸定理）

\[I_{xx}^{(G)} = I_{xx}^{(0)} - My_G^2 = \frac{Mb^2}{6}-\frac{Mb^2}{9} = \frac{Mb^2}{18}\] \[I_{yy}^{(G)} = \frac{Ma^2}{18}, \qquad I_{xy}^{(G)} = -\frac{Mab}{12}+\frac{Mab}{9} = \frac{Mab}{36}\] \[I^{(G)} = \frac{M}{36}\begin{pmatrix}2b^2 & ab \\ ab & 2a^2\end{pmatrix}\]

（4）主慣性モーメント（\(a=b\) の正方形状の場合）

\(a=b\) のとき：

\[\lambda_\pm = \frac{M a^2}{36}(2\pm 1) \quad\Rightarrow\quad \lambda_+ = \frac{Ma^2}{12},\;\lambda_- = \frac{Ma^2}{36}\]

主軸方向はそれぞれ \((1,1)/\sqrt{2}\)（斜め）と \((1,-1)/\sqrt{2}\)。

物理的意味： 主軸方向が「最も慣性が小さい/大きい方向」。人工衛星はこの主軸周りに安定回転する。
解答 2行列の指数関数と回転

（1）べき乗のパターン

\[A^2 = \begin{pmatrix}-\omega^2&0\\0&-\omega^2\end{pmatrix} = -\omega^2 I,\quad A^3 = -\omega^2 A,\quad A^4 = \omega^4 I\]

一般に：\(A^{2k}=(-1)^k\omega^{2k}I,\quad A^{2k+1}=(-1)^k\omega^{2k}A\)

（2）行列指数関数

\begin{align} e^{At} &= \sum_{k=0}^\infty\frac{(-1)^k\omega^{2k}t^{2k}}{(2k)!}I + \frac{1}{\omega}\sum_{k=0}^\infty\frac{(-1)^k\omega^{2k+1}t^{2k+1}}{(2k+1)!}A \\ &= \cos(\omega t)\,I + \frac{\sin(\omega t)}{\omega}\,A \\ &= \begin{pmatrix}\cos\omega t & -\sin\omega t\\\sin\omega t & \cos\omega t\end{pmatrix}\quad\checkmark \end{align}

（3）物理的意味

\[\mathbf{x}(t) = e^{At}\begin{pmatrix}x_0\\0\end{pmatrix} = \begin{pmatrix}x_0\cos\omega t\\x_0\sin\omega t\end{pmatrix}\]

半径 \(x_0\)、角速度 \(\omega\) の 等速円運動。\(A\) は反時計回り回転の生成子。

（4）固有分解

固有値 \(\pm i\omega\)、固有ベクトル \(\mathbf{v}_\pm = (1, \mp i)^T/\sqrt{2}\)。

\[A = P\begin{pmatrix}i\omega&0\\0&-i\omega\end{pmatrix}P^{-1} \;\Rightarrow\; e^{At} = P\begin{pmatrix}e^{i\omega t}&0\\0&e^{-i\omega t}\end{pmatrix}P^{-1}\]

実部・虚部を分離すると同じ結果。

Lie群との関係： \(e^{At}\) はSO(2)の元。\(A\) はそのLie代数 \(\mathfrak{so}(2)\) の元（生成子）。
解答 3射影演算子とスペクトル分解

（1）固有値・固有ベクトル

\[\det(H-\lambda I)=\lambda^2-5\lambda=0 \;\Rightarrow\; \lambda_1=0,\;\lambda_2=5\] \[|\phi_1\rangle = \frac{1}{\sqrt{5}}\begin{pmatrix}2\\-1\end{pmatrix},\quad |\phi_2\rangle = \frac{1}{\sqrt{5}}\begin{pmatrix}1\\2\end{pmatrix}\]

（2）射影演算子

\[P_1 = \frac{1}{5}\begin{pmatrix}4&-2\\-2&1\end{pmatrix},\quad P_2 = \frac{1}{5}\begin{pmatrix}1&2\\2&4\end{pmatrix}\]

（3）性質の確認

\[P_1+P_2 = \frac{1}{5}\begin{pmatrix}5&0\\0&5\end{pmatrix} = I\;\checkmark,\quad P_1 P_2 = 0\;\checkmark\] \[H = 0\cdot P_1 + 5\cdot P_2 = \begin{pmatrix}1&2\\2&4\end{pmatrix}\;\checkmark\]

（4）測定確率

\[\langle\phi_1|\psi\rangle = \frac{1}{\sqrt{5}}(2,\,-1)\begin{pmatrix}1\\0\end{pmatrix} = \frac{2}{\sqrt{5}}\] \[P(\lambda_1) = \left|\frac{2}{\sqrt{5}}\right|^2 = \frac{4}{5}\]
Born則： 測定で \(\lambda_2=5\) が得られる確率は \(1/5\)。量子力学の基本原理。
解答 4二次形式と安定性の判定

（1）行列表示

\[K = \begin{pmatrix}3&1\\1&3\end{pmatrix}\]

（2）正定値判定

第1主小行列式：\(3>0\)。第2主小行列式（行列式）：\(9-1=8>0\)。よって \(V\) は 正定値 → 平衡点は安定。

（3）対角化

\[\det(K-\lambda I)=(\lambda-3)^2-1=0\;\Rightarrow\;\lambda_1=2,\;\lambda_2=4\]

固有ベクトル：\(\mathbf{e}_1=\dfrac{1}{\sqrt{2}}(1,-1)^T\)，\(\mathbf{e}_2=\dfrac{1}{\sqrt{2}}(1,1)^T\)

正規座標 \(\xi_1=(q_1-q_2)/\sqrt{2},\;\xi_2=(q_1+q_2)/\sqrt{2}\) で：

\[V = 2\xi_1^2 + 4\xi_2^2\]

（4）正規振動数

運動方程式 \(\ddot\xi_i = -2\lambda_i\xi_i\)：

\[\omega_1 = \sqrt{2\lambda_1} = 2,\quad \omega_2 = \sqrt{2\lambda_2} = 2\sqrt{2}\]
正規モードの意義： 正規座標系では運動方程式が完全に分離し、各 \(\xi_i\) が独立な調和振動子になる。分子の振動スペクトル解析の基礎。
