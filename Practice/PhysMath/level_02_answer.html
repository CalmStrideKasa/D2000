解答 1単振り子の線形化と非線形補正

（1）線形化

\(\sin\theta\approx\theta\) で \(\ddot{\theta}+\omega_0^2\theta=0\)（\(\omega_0=\sqrt{g/L}\)）。一般解：

\[\theta(t) = C_1\cos\omega_0 t + C_2\sin\omega_0 t, \qquad T_0 = \frac{2\pi}{\omega_0} = 2\pi\sqrt{\frac{L}{g}}\]

（2）特解（\(\theta(0)=\theta_0,\;\dot\theta(0)=0\)）

\[C_1=\theta_0,\;C_2=0 \quad\Rightarrow\quad \theta(t) = \theta_0\cos\omega_0 t\]

（3）非線形補正

\(\theta=\theta_0\cos\omega_0 t + \varepsilon\theta_1\) を代入し、\(\sin\theta\approx\theta-\theta^3/6\)：

\[\ddot\theta_1 + \omega_0^2\theta_1 = \frac{\omega_0^2}{6}\theta_0^3\cos^3\omega_0 t = \frac{\omega_0^2\theta_0^3}{6}\cdot\frac{3\cos\omega_0 t+\cos 3\omega_0 t}{4}\]

右辺の \(\cos\omega_0 t\) 項は強制共鳴項（久期項）なので係数をゼロにするため \(\omega\) を修正する。正確な計算から：

\[\omega = \omega_0\left(1 - \frac{\theta_0^2}{16}\right) \quad\Rightarrow\quad T = T_0\left(1+\frac{\theta_0^2}{16}\right)\]
物理的意味： 振幅が大きいほど周期が長くなる（非調和性）。\(\theta_0=10°\approx0.175\)rad では約 0.19% の補正。
解答 2減衰振動の3つのレジーム

（1）過減衰 \(\gamma>\omega_0\)

特性根 \(\lambda_{1,2} = -\gamma \pm \sqrt{\gamma^2-\omega_0^2}\) （両方実数・負）。一般解：

\[x(t) = C_1 e^{\lambda_1 t} + C_2 e^{\lambda_2 t}\]

初期条件から \(C_1+C_2=x_0,\;C_1\lambda_1+C_2\lambda_2=0\)：

\[C_{1,2} = \frac{\mp\lambda_{2,1}}{\lambda_1-\lambda_2}x_0\]

（2）臨界減衰 \(\gamma=\omega_0\)

重根 \(\lambda=-\gamma\)。一般解 \(x=(C_1+C_2 t)e^{-\gamma t}\)。初期条件から \(C_1=x_0,\;C_2=\gamma x_0\)：

\[x(t) = x_0(1+\gamma t)e^{-\gamma t}\]

過減衰では遅い指数成分が長く残り、不足減衰では振動しながら収束する。臨界減衰は振動なしで最速収束。

（3）不足減衰 \(\gamma<\omega_0\)

\[x(t) = e^{-\gamma t}(C_1\cos\omega_1 t + C_2\sin\omega_1 t) = A_0 e^{-\gamma t}\cos(\omega_1 t - \phi)\]

包絡線 \(A_0 e^{-\gamma t}\) が振幅減衰を表す。

（4）品質因数 \(Q\)

\(N\) サイクル後 \((t=N T_1, T_1=2\pi/\omega_1)\)、エネルギー：

\[E(N T_1) = E_0 e^{-2\gamma N T_1} \approx E_0 e^{-2\gamma N \cdot 2\pi/\omega_0} = E_0\exp\left(-\frac{2\pi N}{Q}\right)\]

\(N=Q\) サイクル後 \(E = E_0 e^{-2\pi}\)。

高Q系（Q=1000など）はエネルギーを長く保持。原子時計・光共振器・NMRの設計に直結。
解答 3強制振動と共鳴

（1）複素法による特解

\(\tilde{x}_p = \tilde{C}e^{i\omega t}\) として代入 \((-\omega^2+2i\gamma\omega+\omega_0^2)\tilde{C}=F_0/m\)：

\[\tilde{C} = \frac{F_0/m}{(\omega_0^2-\omega^2)+2i\gamma\omega}\]

絶対値と位相を分離：

\[A(\omega) = |\tilde{C}| = \frac{F_0/m}{\sqrt{(\omega_0^2-\omega^2)^2+4\gamma^2\omega^2}}\] \[\tan\delta = \frac{2\gamma\omega}{\omega_0^2-\omega^2}, \qquad x_p = A(\omega)\cos(\omega t-\delta)\quad\checkmark\]

（2）共鳴振動数

\(dA/d\omega=0\) より：

\[\omega_{\rm res} = \sqrt{\omega_0^2 - 2\gamma^2}\]

（\(\gamma\to 0\) では \(\omega_{\rm res}\to\omega_0\)。）

（3）非減衰共鳴

\(\gamma=0,\;\omega=\omega_0\) では特性方程式の根と駆動周波数が一致するため通常法が使えない。特解として \(x_p=t(A\cos\omega_0 t+B\sin\omega_0 t)\) を仮定し代入：

\[(-2A\omega_0\sin\omega_0 t + 2B\omega_0\cos\omega_0 t) = \frac{F_0}{m}\cos\omega_0 t\] \[\therefore B=\frac{F_0}{2m\omega_0},\;A=0 \quad\Rightarrow\quad x_p = \frac{F_0}{2m\omega_0}t\sin\omega_0 t\quad\checkmark\]
物理的意味： 振幅が \(t\) に比例して増大 → 構造物の共鳴破壊（Tacoma橋崩壊など）の数学的根拠。
解答 4連立ODE — 連成振動子の正規モード

（1）行列形式

\[K = \begin{pmatrix}k+\kappa & -\kappa \\ -\kappa & k+\kappa\end{pmatrix}\]

（2）固有値問題

\[\det(K - m\omega^2 I) = (k+\kappa-m\omega^2)^2 - \kappa^2 = 0\] \[m\omega^2 = (k+\kappa)\pm\kappa\] \[\omega_- = \sqrt{\frac{k}{m}}, \qquad \omega_+ = \sqrt{\frac{k+2\kappa}{m}}\]

（3）固有モード

\(\omega_-\)：\(K\mathbf{a} = k\mathbf{a}\) より \(\mathbf{a}_- = \dfrac{1}{\sqrt{2}}(1,1)^T\) ← 同相モード（2質点が同方向に動く、中央バネ伸縮なし）

\(\omega_+\)：\(\mathbf{a}_+ = \dfrac{1}{\sqrt{2}}(1,-1)^T\) ← 逆相モード（2質点が逆方向、中央バネが最大伸縮）

正規座標： \(q_1=(x_1+x_2)/\sqrt{2},\;q_2=(x_1-x_2)/\sqrt{2}\) とすると方程式が完全に分離し、各 \(q_i\) が独立な調和振動子になる。これが「正規モード」の意味。
