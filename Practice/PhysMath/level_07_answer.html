解答 1Schwarzschild計量と測地線

（1）保存量の導出

計量が \(t\) に依存しないので、\(\partial_t g_{\mu\nu}=0\)。Killing ベクトル \(k^\mu=(1,0,0,0)\) に対し \(E = -g_{\mu\nu}k^\mu u^\nu = -(g_{tt}\dot{t}) = (1-r_s/r)\dot{t}\)。

同様に \(\varphi\) に依存しないので \(L = g_{\varphi\varphi}\dot{\varphi} = r^2\dot\varphi\)。

（2）有効ポテンシャル

\(g_{\mu\nu}\dot{x}^\mu\dot{x}^\nu = -1\)（\(\theta=\pi/2\)）から：

\[-\left(1-\frac{r_s}{r}\right)\dot{t}^2 + \left(1-\frac{r_s}{r}\right)^{-1}\dot{r}^2 + r^2\dot\varphi^2 = -1\]

\(\dot{t}=E/(1-r_s/r),\;\dot\varphi=L/r^2\) を代入：

\[\dot{r}^2 = E^2 - \left(1-\frac{r_s}{r}\right)\!\left(1+\frac{L^2}{r^2}\right) = E^2 - 1 + \frac{r_s}{r} - \frac{L^2}{r^2} + \frac{r_s L^2}{r^3}\] \[\therefore\; V_{\rm eff} = -\frac{r_s}{2r}+\frac{L^2}{2r^2}-\frac{r_s L^2}{2r^3}\quad\checkmark\]

第3項 \(-r_s L^2/(2r^3)\) はGR固有の補正。\(r\) が小さい（強重力場）ほど効く。この項が水星の近日点移動の原因。

（3）光子と光円

光子は \(ds^2=0\) なので \(\dot{r}^2 = E^2 - (1-r_s/r)L^2/r^2\)。有効ポテンシャル \(V_{\rm ph}=L^2/r^2-r_s L^2/r^3\) の極値（\(dV_{\rm ph}/dr=0\)）：

\[-\frac{2L^2}{r^3}+\frac{3r_s L^2}{r^4}=0 \;\Rightarrow\; r_{\rm ph} = \frac{3r_s}{2}\quad\checkmark\]

この円軌道は不安定（ポテンシャルの最大点）。

（4）重力赤方偏移

半径 \(r\) の静止観測者の固有時：\(d\tau = \sqrt{1-r_s/r}\,dt\)。\(r\to r_s\) で \(d\tau\to0\)（事象の地平面で時間が止まる）。遠方 \((r\to\infty)\) から見ると \(r_s\) 付近の時計は遅れて見える。これが重力赤方偏移で、GPS衛星は毎日約45μs の補正が必要。

Schwarzschild半径： 太陽の場合 \(r_s\approx3\) km。これより内側では逃出速度が \(c\) を超える ＝ ブラックホールの境界。
解答 2Riemann曲率テンソルと重力波

（1）測地線偏差方程式

\(\xi^\mu\)：近傍2測地線の分離ベクトル（「潮汐力で引き伸ばされる方向」）。

\(u^\mu = dx^\mu/d\tau\)：測地線に沿った4速度。

\(R^\mu{}_{\nu\rho\sigma}u^\nu\xi^\rho u^\sigma\)：空間的曲率による加速度 ← GRにおける「重力 ＝ 時空の曲がり」の直接的表現。平坦時空では \(R=0\) なので \(D^2\xi^\mu/d\tau^2=0\)（偏差なし）。

（2）Einstein方程式の各項

\(G_{\mu\nu}\)：幾何学的側（時空の曲率）。\(T_{\mu\nu}\)：物質・エネルギーの分布（エネルギー運動量テンソル）。

意味：「物質は時空を曲げ、時空の曲がりが物質の運動を決める」（Wheeler）。

Bianchi恒等式 \(\nabla^\mu G_{\mu\nu}=0\) は、Einstein方程式と \(\nabla^\mu T_{\mu\nu}=0\)（エネルギー・運動量の保存則）が矛盾なく両立することを保証する。

（3）線形化重力波

\(g_{\mu\nu}=\eta_{\mu\nu}+h_{\mu\nu}\) を Einstein方程式（真空）に代入し1次まで取ると：

\[R_{\mu\nu}^{(1)} = \frac{1}{2}\!\left(-\Box h_{\mu\nu} + \partial_\mu\partial^\lambda h_{\lambda\nu} + \partial_\nu\partial^\lambda h_{\lambda\mu} - \partial_\mu\partial_\nu h\right) = 0\]

ここで \(h=h^\lambda{}_\lambda\)。Lorenzゲージ \(\partial^\mu\bar{h}_{\mu\nu}=0\) を使うと：

\[\Box\bar{h}_{\mu\nu} = 0\quad\checkmark\]

これは光速で伝播する波動方程式 ← 重力波の存在予言（LIGO で2015年初観測）。

重力波の偏極： さらに \(h_{0\mu}=0\)（TT-ゲージ）を選ぶと、\(z\) 方向伝播の重力波はプラス偏極 \(h_{+}\) とクロス偏極 \(h_\times\) の2成分のみ（スピン2）。
解答 3量子調和振動子と生成消滅演算子

（1）交換関係 \([\hat{a},\hat{a}^\dagger]=1\)

\begin{align} [\hat{a},\hat{a}^\dagger] &= \left[\sqrt{\frac{m\omega}{2\hbar}}\hat{q}+\frac{i}{\sqrt{2m\omega\hbar}}\hat{p},\; \sqrt{\frac{m\omega}{2\hbar}}\hat{q}-\frac{i}{\sqrt{2m\omega\hbar}}\hat{p}\right] \\ &= -\frac{i}{2\hbar}[\hat{q},\hat{p}]+\frac{i}{2\hbar}[\hat{p},\hat{q}] = -\frac{i}{2\hbar}(i\hbar) + \frac{i}{2\hbar}(-i\hbar) = \frac{1}{2}+\frac{1}{2} = 1\quad\checkmark \end{align}

（2）\(\hat{H}=\hbar\omega(\hat{N}+1/2)\)

\[\hat{a}^\dagger\hat{a} = \frac{m\omega}{2\hbar}\hat{q}^2 + \frac{1}{2m\omega\hbar}\hat{p}^2 + \frac{i}{2\hbar}[\hat{q},\hat{p}] = \frac{\hat{H}}{\hbar\omega}-\frac{1}{2}\] \[\therefore\quad\hat{H} = \hbar\omega\!\left(\hat{a}^\dagger\hat{a}+\frac{1}{2}\right)\quad\checkmark\]

（3）昇降演算子の作用

\([\hat{N},\hat{a}]=[\hat{a}^\dagger\hat{a},\hat{a}]=-\hat{a}\)（交換関係から）より：

\[\hat{N}(\hat{a}|n\rangle) = (\hat{a}\hat{N}-\hat{a})|n\rangle = (n-1)\hat{a}|n\rangle\]

つまり \(\hat{a}|n\rangle \propto |n-1\rangle\)。規格化：\(\|\hat{a}|n\rangle\|^2=\langle n|\hat{a}^\dagger\hat{a}|n\rangle=n\) なので \(\hat{a}|n\rangle=\sqrt{n}|n-1\rangle\)。同様に \(\hat{a}^\dagger|n\rangle=\sqrt{n+1}|n+1\rangle\)。

また \(n\ge0\) の整数のみ許容（\(n-1\ge0\) が繰り返し成り立つには \(n=0\) での \(\hat{a}|0\rangle=0\) で止まる必要がある）。

（4）コヒーレント状態

\(|\alpha\rangle=\sum c_n|n\rangle\) とし \(\hat{a}|\alpha\rangle=\alpha|\alpha\rangle\) を使うと \(\sqrt{n}c_n=\alpha c_{n-1}\)：

\[c_n = \frac{\alpha^n}{\sqrt{n!}}c_0\]

規格化 \(\langle\alpha|\alpha\rangle=1\) から \(|c_0|^2\sum|\alpha|^{2n}/n! = |c_0|^2 e^{|\alpha|^2}=1\)、よって \(c_0=e^{-|\alpha|^2/2}\)：

\[|\alpha\rangle = e^{-|\alpha|^2/2}\sum_{n=0}^\infty\frac{\alpha^n}{\sqrt{n!}}|n\rangle\quad\checkmark\]

光子数 \(n\) の確率：\(P(n)=|c_n|^2=e^{-|\alpha|^2}\dfrac{|\alpha|^{2n}}{n!}\) ← 平均 \(\bar{n}=|\alpha|^2\) のPoisson分布 ✓

コヒーレント状態： レーザー光の量子状態の記述に使われる「最も古典的な」量子状態。最小不確定性 \(\Delta q\Delta p=\hbar/2\) を常に満たす。
解答 4Poisson括弧と正準量子化・経路積分の芽

（1）Poisson括弧の性質

(a) \(\{q,p\}=\partial q/\partial q\cdot\partial p/\partial p - \partial q/\partial p\cdot\partial p/\partial q = 1\cdot1-0\cdot0=1\quad\checkmark\)

(b) \(\{q,H\}=\partial H/\partial p = p/m = \dot{q}\quad\checkmark\)，\(\{p,H\}=-\partial H/\partial q = -V'(q)=\dot{p}\quad\checkmark\)

(c) \(\{L_i,L_j\} = \varepsilon_{ijk}L_k\)：インデックス計算で \(\{L_1,L_2\}=\{q_2p_3-q_3p_2,\;q_3p_1-q_1p_3\}=q_2p_1-q_1p_2=L_3\) ✓

（2）Diracの量子化条件

Diracの対応：\(\{f,g\}_{\rm classical} \;\to\; \dfrac{1}{i\hbar}[\hat{f},\hat{g}]_{\rm quantum}\)

したがって \(\{q,p\}=1 \;\to\; [\hat{q},\hat{p}]=i\hbar\)。

角運動量：\(\{L_i,L_j\}=\varepsilon_{ijk}L_k \;\to\; [\hat{L}_i,\hat{L}_j]=i\hbar\varepsilon_{ijk}\hat{L}_k\)。これが角運動量演算子の代数（su(2) Lie代数）の起源。

（3）経路積分の構成

1ステップ：

\[\langle q_{j+1}|e^{-i\hat{H}\varepsilon/\hbar}|q_j\rangle \approx \int\frac{dp_j}{2\pi\hbar}\langle q_{j+1}|p_j\rangle\langle p_j|e^{-iH\varepsilon/\hbar}|q_j\rangle\] \[\approx \int\frac{dp_j}{2\pi\hbar}\exp\!\left(\frac{i}{\hbar}\left[p_j(q_{j+1}-q_j) - \varepsilon H(q_j,p_j)\right]\right)\]

\(H=p^2/(2m)+V(q)\) のとき \(p\) 積分（Gauss積分）を実行し：

\[\langle q_{j+1}|e^{-i\hat{H}\varepsilon/\hbar}|q_j\rangle \approx \sqrt{\frac{m}{2\pi i\hbar\varepsilon}}\exp\!\left(\frac{i\varepsilon}{\hbar}\left[\frac{m}{2}\left(\frac{q_{j+1}-q_j}{\varepsilon}\right)^2 - V(q_j)\right]\right)\]

全ステップを掛け合わせると \(\varepsilon\to0\) の極限で：

\[\langle q_f|e^{-i\hat{H}t/\hbar}|q_i\rangle = \int\mathcal{D}[q]\exp\!\left(\frac{i}{\hbar}\int_0^t L(q,\dot{q})dt\right)\quad\checkmark\]

ここで括弧内の指数は作用汎関数 \(S[q]=\int L\,dt\)。

Feynman経路積分の洞察： 量子粒子は「起こりうるすべての経路」を同時に通過し、各経路の寄与 \(e^{iS/\hbar}\) を重ね合わせる。古典極限 \(\hbar\to0\) では定常位相近似から最小作用の経路（古典軌道）のみが残る。
