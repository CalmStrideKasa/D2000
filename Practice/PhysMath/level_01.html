問題 1 スカラー場の勾配と方向微分
物理的文脈：温度場や電位場の空間変化を記述する。

スカラー場 \(\phi(x,y,z) = x^2 y + y z^2\) について、以下を求めよ。

（1）

点 \(P = (1, 2, -1)\) における勾配 \(\nabla\phi\Big|_P\) を求めよ。

（2）

点 \(P\) において、方向 \(\mathbf{n} = \dfrac{1}{\sqrt{3}}(1,1,1)\) への方向微分 \(D_\mathbf{n}\phi\) を求めよ。

（3）

\(\phi\) が最も急激に増加する方向と、そのときの変化率を求めよ。

ヒント： 方向微分は \(D_\mathbf{n}\phi = \nabla\phi\cdot\mathbf{n}\)。最大変化率は \(|\nabla\phi|\)。
問題 2 電場の発散とGaussの法則
物理的文脈：Gaussの法則 \(\nabla\cdot\mathbf{E} = \rho/\varepsilon_0\) により電荷分布と電場を結ぶ。

電場 \(\mathbf{E}(x,y,z) = \left(2xz,\,-y^2,\,y^2 z\right)\) について、以下を求めよ。

（1）

発散 \(\nabla\cdot\mathbf{E}\) を計算せよ。

（2）

Gaussの法則 \(\nabla\cdot\mathbf{E} = \rho/\varepsilon_0\) を用いて、点 \((1,1,1)\) における電荷密度 \(\rho\) を求めよ。

（3）

辺長 2 の立方体領域 \(V = \{0\le x,y,z\le 2\}\) におけるGaussの定理： \[\oiint_S \mathbf{E}\cdot d\mathbf{S} = \iiint_V \nabla\cdot\mathbf{E}\,dV\] を直接体積積分を計算することで確認せよ。

ヒント： (3) では体積積分 \(\iiint_V (2z - 2y + y^2)\,dV\) を各変数で順に積分する。
問題 3 磁場の回転とStokesの定理
物理的文脈：アンペール（準定常）の法則 \(\nabla\times\mathbf{B} = \mu_0\mathbf{J}\)。

ベクトル場 \(\mathbf{F}(x,y,z) = \left(-y,\,x,\,0\right)\) について、以下を求めよ。

（1）

回転 \(\nabla\times\mathbf{F}\) を計算せよ。

（2）

\(xy\) 平面上の半径 \(R\) の円盤 \(S: x^2+y^2\le R^2,\; z=0\) に対してStokesの定理を用い： \[\oint_C \mathbf{F}\cdot d\mathbf{l} = \iint_S (\nabla\times\mathbf{F})\cdot d\mathbf{S}\] の両辺を直接計算し、一致することを確認せよ。

（3）

この \(\mathbf{F}\) が一様な角速度 \(\omega\) で回転する剛体の速度場 \(\mathbf{v} = \boldsymbol{\omega}\times\mathbf{r}\)（\(\boldsymbol{\omega} = \omega\mathbf{e}_z\)）と一致することを示し、\(\nabla\times\mathbf{v} = 2\boldsymbol{\omega}\) を確認せよ。

ヒント： (2) 左辺：円周上で \(x = R\cos\theta,\, y = R\sin\theta\) とパラメータ化。
問題 4 重力ポテンシャルとLaplacian
物理的文脈：点質量の重力ポテンシャルと Poisson 方程式 \(\nabla^2\phi = 4\pi G\rho\)。

\(r = \sqrt{x^2+y^2+z^2}\)（\(r > 0\)）として、ポテンシャル \(\phi = -\dfrac{GM}{r}\) について答えよ。

（1）

\(\nabla\phi\) を計算し、重力加速度 \(\mathbf{g} = -\nabla\phi\) が \(\mathbf{g} = -\dfrac{GM}{r^3}\mathbf{r}\) であることを示せ。

（2）

\(r > 0\) の領域でのLaplacian \(\nabla^2\phi\) を計算し、真空中のLaplace方程式 \(\nabla^2\phi = 0\) が満たされることを示せ。

\(r\)の偏微分：\(\dfrac{\partial r}{\partial x} = \dfrac{x}{r}\)，球対称関数のLaplacian：\(\nabla^2 f(r) = f''(r) + \dfrac{2}{r}f'(r)\)

（3）

\(r^n\) 型の関数 \(f = r^n\) に対して \(\nabla^2(r^n) = n(n+1)r^{n-2}\) を示せ。

ヒント： \(\nabla^2(1/r) = 0\) は (2) の特殊ケース（\(n = -1\)）で確認できる（\(r>0\)）。
