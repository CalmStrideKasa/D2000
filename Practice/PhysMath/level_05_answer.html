解答 1Levi-Civitaシンボルとベクトル恒等式

（1）BAC-CAB則

\begin{align} [(\mathbf{A}\times\mathbf{B})\times\mathbf{C}]^i &= \varepsilon^{ijk}(\varepsilon_{jlm}A^l B^m)C_k = \varepsilon^{ijk}\varepsilon_{jlm}A^l B^m C_k \\ &= -\varepsilon^{ijk}\varepsilon_{ljm}A^l B^m C_k = -(\delta^i_l\delta^k_m - \delta^i_m\delta^k_l)A^l B^m C_k \\ &= -A^i B^m C_m + B^i A^l C_l = B^i(\mathbf{A}\cdot\mathbf{C}) - A^i(\mathbf{B}\cdot\mathbf{C})\quad\checkmark \end{align}

（2）\(\nabla\times(\nabla\times\mathbf{F})\)

\[[\nabla\times(\nabla\times\mathbf{F})]^i = \varepsilon^{ijk}\partial_j(\varepsilon_{klm}\partial_l F_m) = \varepsilon^{ijk}\varepsilon_{klm}\partial_j\partial_l F_m\] \[= (\delta^i_l\delta^j_m - \delta^i_m\delta^j_l)\partial_j\partial_l F_m = \partial_l\partial^i F^l - \partial_j\partial^j F^i = \nabla^i(\nabla\cdot\mathbf{F}) - \nabla^2 F^i\quad\checkmark\]

（3）電磁波の波動方程式

\[\nabla\times(\nabla\times\mathbf{E}) = \nabla(\underbrace{\nabla\cdot\mathbf{E}}_{=0}) - \nabla^2\mathbf{E} = -\nabla^2\mathbf{E}\]

Faradayの法則 \(\nabla\times\mathbf{E}=-\partial\mathbf{B}/\partial t\)、Ampereの法則 \(\nabla\times\mathbf{B}=\mu_0\varepsilon_0\partial\mathbf{E}/\partial t\) を使うと：

\[-\nabla^2\mathbf{E} = -\frac{\partial}{\partial t}(\nabla\times\mathbf{B}) = -\mu_0\varepsilon_0\frac{\partial^2\mathbf{E}}{\partial t^2} \;\Rightarrow\; \nabla^2\mathbf{E} = \mu_0\varepsilon_0\frac{\partial^2\mathbf{E}}{\partial t^2}\quad\checkmark\]
光速 \(c=1/\sqrt{\mu_0\varepsilon_0}\) が導出される。Maxwell方程式の最大の成果のひとつ。
解答 2電磁場テンソルとLorentz変換

（1）\(F^{\mu\nu}\) の成分

\(A^\mu=(\phi/c, A_x, A_y, A_z)\)、\(\partial^\mu = (-\partial_t/c, \nabla)\) を使い \(F^{01}=\partial^0 A^1-\partial^1 A^0\)：

\[F^{01} = -\frac{1}{c}\frac{\partial A_x}{\partial t} - \frac{\partial}{\partial x}\frac{\phi}{c} = -\frac{E_x}{c}\quad\checkmark\]

磁場成分：\(F^{12} = \partial^1 A^2 - \partial^2 A^1 = \partial_x A_y - \partial_y A_x = -B_z\quad\checkmark\)

（2）Lorentz力の再現

\(u^\nu \approx (c, v_x, v_y, v_z)\)（\(v\ll c\)）として空間成分 \(f^i = q F^{i\nu}u_\nu\)：

\[f^1 = q(F^{10}u_0 + F^{12}u_2 + F^{13}u_3) = q\left(\frac{E_x}{c}\cdot c + (-B_z)v_y + B_y v_z\right) = q(E_x + v_y B_z - v_z B_y)\]

これは \([\mathbf{v}\times\mathbf{B}]_x = v_y B_z - v_z B_y\) を含むので \(\mathbf{f}=q(\mathbf{E}+\mathbf{v}\times\mathbf{B})\) ✓

（3）Lorentz変換による場の変換

\(x\) 方向ブーストで \(F'^{01}=\Lambda^0{}_\alpha\Lambda^1{}_\beta F^{\alpha\beta}\) を計算すると：

\[E'_x = E_x,\quad E'_y = \gamma(E_y - \beta c B_z),\quad E'_z = \gamma(E_z + \beta c B_y)\] \[B'_x = B_x,\quad B'_y = \gamma\!\left(B_y + \frac{\beta}{c}E_z\right),\quad B'_z = \gamma\!\left(B_z - \frac{\beta}{c}E_y\right)\]
洞察： 電場と磁場は別々の「物理量」ではなく、同一の電磁場テンソルの、座標系依存の表れに過ぎない。
解答 3球面座標のChristoffel記号と測地線

（1）非零Christoffel記号

\[\Gamma^r_{\theta\theta}=-r,\quad\Gamma^r_{\varphi\varphi}=-r\sin^2\theta\] \[\Gamma^\theta_{r\theta}=\Gamma^\theta_{\theta r}=\frac{1}{r},\quad\Gamma^\theta_{\varphi\varphi}=-\sin\theta\cos\theta\] \[\Gamma^\varphi_{r\varphi}=\Gamma^\varphi_{\varphi r}=\frac{1}{r},\quad\Gamma^\varphi_{\theta\varphi}=\Gamma^\varphi_{\varphi\theta}=\cot\theta\]

（2）球面上の測地線（\(r=\mathrm{const}\)）

\(\dot r=\ddot r=0\) として \(\theta,\varphi\) の方程式：

\[\ddot\theta - \sin\theta\cos\theta\,\dot\varphi^2 = 0\] \[\ddot\varphi + 2\cot\theta\,\dot\theta\dot\varphi = 0\]

解は大円（赤道面の回転）。

（3）加速度の \(\theta\) 成分

\[a^\theta = \ddot\theta + 2\cdot\frac{1}{r}\dot r\dot\theta - \sin\theta\cos\theta\,(\dot\varphi)^2\]

（球面極座標での加速度公式に一致）

測地線方程式は「慣性系での直線運動」の曲線座標版。一般相対論では重力のない時空での自由落下軌道を記述する。
解答 4応力テンソルと主応力

（1）トラクションベクトル

\[\mathbf{t} = \sigma\cdot\hat{n} = \frac{1}{\sqrt{2}}\begin{pmatrix}10+5\\5+3\\0+2\end{pmatrix} = \frac{1}{\sqrt{2}}\begin{pmatrix}15\\8\\2\end{pmatrix}\;\text{[MPa]}\]

（2）法線・せん断応力

\[\sigma_n = \mathbf{t}\cdot\hat{n} = \frac{1}{\sqrt{2}}(15,8,2)\cdot\frac{1}{\sqrt{2}}(1,1,0) = \frac{15+8}{2} = 11.5\;\text{MPa}\] \[|\boldsymbol{\tau}| = \sqrt{|\mathbf{t}|^2 - \sigma_n^2} = \sqrt{\frac{15^2+8^2+2^2}{2}-11.5^2} = \sqrt{\frac{293}{2}-132.25} = \sqrt{14.25}\approx 3.78\;\text{MPa}\]

（3）主応力

上左 \(2\times2\) ブロックの固有値：\(\det\begin{pmatrix}10-\lambda&5\\5&3-\lambda\end{pmatrix}=\lambda^2-13\lambda+5=0\)

\[\lambda = \frac{13\pm\sqrt{169-20}}{2} = \frac{13\pm\sqrt{149}}{2}\] \[\lambda_1 \approx 12.60\;\text{MPa},\quad\lambda_2 \approx 0.40\;\text{MPa},\quad\lambda_3 = -1\;\text{MPa}\]

（4）最大せん断応力

\[\tau_{\max} = \frac{\lambda_1-\lambda_3}{2} = \frac{12.60-(-1)}{2} \approx 6.80\;\text{MPa}\]
Mohr円： 主応力の最大・最小値と最大せん断応力は材料の降伏・破壊条件（Von Misesの降伏基準など）の設計に使われる。
